<strong>⚠Arch Linuxプロジェクトとは全く関係ありません！⚠</strong>

Arch系ディストリビューション(Manjaro含む)やMSYS2で、外部リポジトリ(Ubuntu風に言うとPPA)を追加する際に作業時間を短縮するコマンドです。

## 導入方法
```
yay -S add-pacman-repository
```
